# Flask server (Mac)

## 1. 프로젝트 생성 
```bash
mkdir company
cd company

$ 가상환경 생성 및 활성화
python3 -m venv venv
source venv/bin/activate
```

## 2. 설치 (pip upgrade & Flask)
```bash
python3 -m pip install --upgrade pip
python3 -m pip install flask
```

## 3. 실행
```bash
$ vi app.py

from flask import Flask

app = Flask(__name__)

@app.route('/')
def home():
    return "Hello, Flask!"

if __name__ == '__main__':
    app.run(debug=True)

* :wq
```

```bash
python3 app.py
```
